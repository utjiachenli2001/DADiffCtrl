# Trajectory Influence Functions for Diffusion-Based Control

> **NeurIPS 2026 Submission** — Data Attribution for Safe Planning

Trajectory Influence Functions (TIF) attribute the influence of individual training trajectories on plans generated by diffusion-based control. This enables identifying harmful training data, curating datasets, and improving planner safety.

---

## Quick Start

```bash
# Install
conda create -n dadiffctrl python=3.10 && conda activate dadiffctrl
pip install "cython<3" "setuptools<81"
pip install git+https://github.com/aravindr93/mjrl.git
pip install torch>=2.0 numpy scipy scikit-learn tqdm d4rl "gym==0.21.0"

# Run smoke test (~15 min)
python run_grid.py --smoke-test

# Full reproduction (4 GPUs, ~18 hours)
python run_grid.py --experiments all --mode parallel --n-workers 4 --gpu-ids 0 1 2 3
python run_ablation.py
python aggregate_results.py --latex
```

## API

```python
from configs import ExperimentConfig
from diffuser_minimal import TrajectoryDataset, train, plan
from influence_functions import TrajectoryInfluenceComputer
import torch

# Setup
config = ExperimentConfig(env_name="halfcheetah", dataset="medium")
dataset = TrajectoryDataset(env_name="halfcheetah", dataset_variant="medium", horizon=32)
diffusion, _ = train(config, dataset=dataset)

# Generate plan
plans = plan(diffusion, dataset, config)
ref_plan = torch.from_numpy((plans[0] - dataset.data_mean) / dataset.data_std).float()

# Compute influence scores
tic = TrajectoryInfluenceComputer(model=diffusion, dataset=dataset,
                                   config=config.influence,
                                   diffusion_steps=config.diffuser.n_diffusion_steps)
tic.compute_hessian_approximation()
scores = tic.compute_all_influences(ref_plan, proxy_type="likelihood")

# Interpret: positive = helpful, negative = harmful
print(f"Most harmful: {scores.argsort()[:5]}")
print(f"Most helpful: {scores.argsort()[-5:][::-1]}")
```

## CLI Options

| Flag | Values | Description |
|------|--------|-------------|
| `--env` | halfcheetah, hopper, walker2d | D4RL environment |
| `--dataset` | medium, medium-replay, medium-expert | Dataset variant |
| `--experiment` | lds, safety, curation, intervention, all | Evaluation protocol |
| `--hessian-approx` | ekfac, kfac, diagonal, plain_dot | Curvature approximation |
| `--seed` | 0, 1, 2, ... | Random seed |
| `--smoke-test` | — | Minimal validation |
| `--resume` | — | Skip completed cells |
| `--dry-run` | — | Print commands only |
